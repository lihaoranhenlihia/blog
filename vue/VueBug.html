                  # Vue 中遇到的 BUG

                  # 当出现"xxx is defined,but never used"

                  • 在.eslintrc 中的 rules 中加入"'no-unused-vars': 'off'"即可

                  # 当 Vue 组件中，我们一开始没有赋值的时候，后面通过请求来进行普通赋值的时候，我们会发现值虽然赋上了，但是页面没有变化例如 Img 的 src 属性，我们可以通过以下做法来进行明确告诉 Vue 实例，给数据中的某一个属性添加一个新属性

                  • this.$set(this.model, 'icon', value)

                  
* 如果我们不想这麽麻烦的话，我们可以通过在建立组建的时候，就将我们需要使用新值得属性定义好，可以先暂时定义为空或者''

```javascript
data(){
    return {
        model:{
            icon:'',
            icon:null
        }
    }
}
